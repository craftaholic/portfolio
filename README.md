# Tommy Tran's Portfolio

My personal portfolio at [craftaholic.sh](https://craftaholic.sh), built with [Astro](https://astro.build) and [Three.js](https://threejs.org).

## Tech Stack

- **Framework**: Astro 5
- **3D Graphics**: Three.js
- **Runtime**: Bun
- **Deployment**: Cloudflare Pages

## Getting Started

```bash
# Install dependencies
bun install

# Start dev server
bun dev

# Build for production
bun build

# Preview production build
bun preview
```

## Project Structure

```
src/
├── components/     # Astro components
├── content/        # Markdown content (blog, work, products)
├── layouts/        # Page layouts
├── pages/          # Route pages
├── styles/         # Global CSS
└── config.ts       # Site configuration
```
