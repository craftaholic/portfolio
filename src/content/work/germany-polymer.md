## Role
**Platform Engineer** at FPT Software

## Duration
2023

## Overview
Led the migration and automation initiative for a German polymer manufacturing client, moving their entire development workflow from GitLab to GitHub while building shared automation tools to improve developer experience.

## Key Achievements

- **Large-Scale Migration**: Automated migration of **3000+ GitLab repositories** to GitHub, including system and user integration, **completed in under 2 months**
- **Developer Experience**: Built shared automation tools to reduce developer cognitive load across teams
- **IaC Libraries**: Developed AWS CDK construct libraries, speeding up infrastructure setup
- **Integration Workflows**: Created seamless integration workflows using Python, TypeScript, and AWS Lambda

## Technologies
- **Source Control**: GitHub, GitLab
- **Languages**: Python, TypeScript
- **Cloud**: AWS (Lambda, S3, DynamoDB)
- **IaC**: AWS CDK
- **CI/CD**: GitHub Actions, GitLab CI
- **Container Orchestration**: EKS

## Impact
- Successfully migrated 3000+ repositories with zero data loss
- Reduced infrastructure setup time by 70% with CDK construct libraries
- Improved developer onboarding time from 2 weeks to 2 days
- Standardized automation patterns across 50+ development teams
- Created reusable GitHub Actions workflows
