A modern developer portfolio built with Astro 5 and Three.js. Features an interactive low-poly 3D island inspired by craftz.dog, with a terminal/Vim aesthetic throughout.

## Highlights

**Performance**
- Static site generation for instant page loads
- Automatic image optimization with WebP conversion
- Deployed on Cloudflare Pages edge network

**Developer Experience**
- Bun runtime for fast builds
- Content collections for type-safe markdown
- View Transitions for smooth navigation

**Customization**
- 9 built-in color themes
- Configurable navigation and social links
- Easy content management via markdown files
